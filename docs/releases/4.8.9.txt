Version 4.8.9
=============

Released: November 22, 2025


Summary
-------

Version 4.8.9 incorporates incremental improvements and refinements inherited from version 4.7.7, along with several key dependency upgrades. This update focuses on maintaining compatibility with upstream libraries, enhancing security, and improving overall stability across document
processing, scheduling, and AI integration components.


Changes
-------

Merges
^^^^^^

- Integrated improvements and changes introduced in version 4.7.7.


Dependencies
^^^^^^^^^^^^

The following dependencies were updated to their latest compatible versions:

- Markdown: 3.7 to 3.10
- Pillow: 10.4.0 to 12.0.0
- dateparser: 1.2.0 to 1.2.2
- django-celery-beat: 2.6.0 to 2.8.1
- nh3: 0.2.18 to 0.3.2
- ollama: 0.2.1 to 0.6.1
- pypdf: 6.2.0 to 6.3.0
- sentry-sdk: 2.44.0 to 2.45.0

These updates provide improved security, bug fixes, and performance optimizations across document rendering, background scheduling, and AI-assisted metadata extraction features.
