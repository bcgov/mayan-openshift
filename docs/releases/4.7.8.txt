Version 4.7.8
=============

Released: December 13, 2025


Summary
-------

This maintenance release backports security, internationalization, and packaging improvements from the 4.10, 4.9, and 4.8 series. It tightens navigation redirect handling, improves translation coverage across several UI elements, updates core dependencies (including Django and django-celery-beat), and adds small tooling and worker-behavior refinements.


Changes
-------

Backports
^^^^^^^^^

- Backported assorted fixes from the 4.10, 4.9, and 4.8 series.


Storage
^^^^^^^

- Sorted storage-related dependencies to ensure a consistent and deterministic installation order.


Security
^^^^^^^^

- Enforced same-origin and HTTP-only redirection when breaking out of an AJAX content container and during normal single-page application navigation.


Code quality
^^^^^^^^^^^^

- Updated the project .gitignore to exclude __pycache__ folders.


Internationalization
^^^^^^^^^^^^^^^^^^^^

- Made the Smart link condition fields `inclusion` and `operator` fully translatable.
- Made document file search fields translatable.
- Made the tag document list link title translatable.
- Made the task worker labels translatable.
- Added a missing locales app migration.


Packaging
^^^^^^^^^

- Ensured that only Mayan EDMS packages are uploaded to PyPI.


Dependencies
^^^^^^^^^^^^

- Updated Django from version 4.2.26 to 4.2.27 to address CVE-2025-13372 and CVE-2025-64460.
- Updated `django-auth-ldap` from version 4.8.0 to 5.2.0.
- Updated `django-celery-beat` from version 2.6.0 to 2.8.1.


Developer tools
^^^^^^^^^^^^^^^

- Added a Makefile target to clean up all installed Python libraries.


Task manager
^^^^^^^^^^^^

- Ensured the `Worker` class returns string objects for its label content.
