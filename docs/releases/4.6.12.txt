Version 4.6.12
==============

Released: December 14, 2025


Summary
-------

This is a maintenance release in the 4.6 series that backports several fixes, improvements, and dependency updates from versions 4.10, 4.9, 4.8, and 4.7. It focuses on security hardening, translation completeness, build and release workflow improvements, and dependency updates.

Changes
-------

Backported changes
^^^^^^^^^^^^^^^^^^

- Sort storage dependencies for more deterministic behavior.
- Enforce same-origin and HTTP-only redirection rules when breaking from an AJAX content container and during normal single-page application navigation.
- Add `__pycache__/` to `.gitignore`.
- Make the Smart Link condition fields `inclusion` and `operator` fully translatable.
- Make document file search fields translatable.
- Make the tag document list link title translatable.
- Make task worker labels translatable.
- Ensure only Mayan EDMS packages are uploaded to PyPI.
- Update internal dependencies:

  - Django from 4.2.26 to 4.2.27 to address CVE-2025-13372 and CVE-2025-64460.
  - `django-auth-ldap` from 4.8.0 to 5.2.0.

- Move Python-related makefile targets into their own dedicated makefile.
- Add a makefile target to clean up all Python libraries.
- Ensure the `Worker` class returns string objects for its label content.
- Update `pytz` from 2024.1 to 2025.2.
- Synchronize locale app migrations (0009-0012) with version 4.7.


Dependencies
^^^^^^^^^^^^

- Update `django-celery-beat` from 2.5.0 to 2.8.1.
- Update `pip` from 25.2 to 25.3.
