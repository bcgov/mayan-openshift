Version 4.7.7
=============

Released: November 14, 2025


Summary
-------

Version 4.7.7 integrates fixes and improvements from version 4.6.11, and applies critical security updates to dependencies.


Changes
-------

Dependency Updates
^^^^^^^^^^^^^^^^^^

In addition to the version updates carried over from version 4.6.11, this release adds:

- django-model-utils from 4.5.1 to 5.0.0
- pypdf from 6.1.3 to 6.2.0
- requests from 2.32.4 to 2.32.5
- sentry-sdk from 2.43.0 to 2.44.0
