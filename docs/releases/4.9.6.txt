Version 4.9.6
=============

Released: December 1, 2025

Summary
-------

Version 4.9.6 delivers structural improvements to form handling,
adds a new makefile maintenance target, and updates several core dependencies to enhance
stability, performance, and security. This release also consolidates dynamic field reload
logic, improving maintainability for form-based components.


Changes
-------

Core Enhancements
^^^^^^^^^^^^^^^^^

- Renamed the file metadata model mixin class to improve clarity and maintain consistency with other model mixin conventions.
- Consolidated dynamic filtering and field reload logic under a new mixin:
  ``FormMixinFilteredFieldsReload``, simplifying reusable form field management.
- Integrated cumulative improvements and fixes from version 4.8.9.


Build and Maintenance
^^^^^^^^^^^^^^^^^^^^^

- Added a new makefile target to clean up all Python libraries, simplifying maintenance of development environments.


Dependency Updates
^^^^^^^^^^^^^^^^^^

The following dependency upgrades improve security and runtime efficiency:

- coverage: 6.5.0 to 7.12.0
- coverall: 3.3.1 to 4.0.2
- django-test-migrations: 1.4.0 to 1.5.0
- gevent: 24.11.1 to 25.9.1 (addresses PVE-2025-76839)
- greenlet: 3.1.1 to 3.2.4
- pip: 25.2 to 25.3
- pypdf: 6.3.0 to 6.4.0
- safety: 3.2.13 to 3.7.0
- sentry-sdk: 2.45.0 to 2.46.0


Appearance App
^^^^^^^^^^^^^^

- Added the path ``appearance/node_modules/bootswatch/docs/*`` to the
  ``static_media_ignore_patterns`` list to prevent unnecessary static file collection
  from third-party theme documentation assets.
