Version 4.9.7
=============

Released: December 13, 2025

Summary
-------

Backported changes from the 4.10 series improve security, internationalization, and packaging consistency. This release also includes important dependency updates addressing known vulnerabilities.

Changes
-------

Backports
^^^^^^^^^

- Enforced same-origin and HTTP-only redirection when breaking from an AJAX content container and during normal single page application navigation.
- Added `__pycache__` folders to the project's `.gitignore` file.
- Made the Smart Link condition fields `inclusion` and `operator` fully translatable.
- Made document file search fields translatable.
- Made the tag document list link title translatable.
- Made the task worker labels translatable.
- Added a missing `locales` app migration.
- Ensured that only Mayan EDMS packages are uploaded to PyPI.
- Fixed appearance settings variable names.


Dependencies
^^^^^^^^^^^^

- Updated Django from version 4.2.26 to 4.2.27 to address CVE-2025-13372 and CVE-2025-64460.
- Updated `django-auth-ldap` from version 5.1.0 to 5.2.0.
