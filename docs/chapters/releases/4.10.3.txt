Version 4.10.3
==============

Released: December 24, 2025

Summary
-------

This is a maintenance release that includes fixes, improvements, and dependency updates. It also incorporates all changes from the versions 4.9.7 and 4.9.8.


Changes
-------

- Updated the document upload wizard logic to automatically skip the Cabinet and Tag steps when the user does not have access to any cabinets or tags.
- Aligned the permission checks for the Cabinet and Tag wizard steps with the existing logic used by the Metadata wizard steps.
- Modernized the `PythonDependency` class version checking mechanism by removing the use of the deprecated `pkg_resources` library.

Fixes
-----

- Worked around an undocumented backward-incompatible behavior introduced in `django-formtools` version 2.4.0 that affected wizard functionality.

Dependencies
------------

- Updated `django-formtools` from version 2.3 to 2.5.1.
- Updated `pypdf` from version 6.4.2 to 6.5.0.
