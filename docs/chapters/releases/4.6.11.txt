Version 4.6.11
==============

Released: November 8, 2025


Summary
-------

This release focuses on broadening internationalization support, updating
core dependencies, and improving test structure and reliability.

A total of thirteen new languages and regional variants have been added,
expanding Mayan EDMS's accessibility across Europe, Asia, and the Middle East.

The release also brings numerous dependency upgrades for security and
compatibility, refined internal test organization, and improved environment
variable handling.


Changes
-------


Internationalization
^^^^^^^^^^^^^^^^^^^^

Thirteen new languages and regional variants were added to enhance multilingual support:

- Armenian (Armenia)
- Bulgarian (Bulgaria)
- French (France)
- German
- Greek (Greece)
- Hungarian (Slovakia)
- Hungarian (Hungary)
- Japanese
- Persian (Iran)
- Russian (Russia)
- Spanish (Ecuador)
- Spanish (Spain)
- Tibetan
- Ukrainian (Ukraine)

These additions significantly expand localization coverage, bringing native
language support to new regions and improving usability for non-English users.

Testing & Development
^^^^^^^^^^^^^^^^^^^^^

- Split the file metadata app test suite into separate modules for views and
  mixins, simplifying maintenance and readability.

- Adjusted checkout tests to silence document indexing errors that could
  appear during automated testing, avoiding confusion in CI logs.

- Added logic to retrieve the `psutil` version directly from the `config.env`
  file, ensuring that version consistency is automatically maintained across
  environments.
