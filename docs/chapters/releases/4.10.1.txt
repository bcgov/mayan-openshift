4.10.1
======

Released: December 8, 2025


Summary
-------

Version 4.10.1 includes improvements from version 4.9.6, dependency updates, test environment refinements, and CI pipeline adjustments. This release also addresses several security advisories through upgraded packages.

Changes
-------

General
^^^^^^^

- Improvements and changes from version 4.9.6.
- Fixed appearance settings variable names.

Dependencies
^^^^^^^^^^^^
- Updated Django from version 4.2.26 to 4.2.27 to address CVE-2025-13372 and CVE-2025-64460.
- Updated ``django-auth-ldap`` from version 5.1.0 to 5.2.0.
- Updated ``openai`` from version 1.107.2 to 1.109.1.

Continuous Integration
^^^^^^^^^^^^^^^^^^^^^^

- Removed redundant ``apt-get update`` invocation in GitLab CI.
- Ensured only Mayan EDMS packages are uploaded to PyPI.

Testing
^^^^^^^
- Disabled REST API throttling when running the unit tests to ensure consistent test behavior.
