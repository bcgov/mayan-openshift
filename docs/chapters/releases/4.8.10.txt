Version 4.8.10
==============

Released: December 13, 2025


Summary
-------

This release backports several improvements from the 4.10 and 4.9 series, introducing security hardening, translation updates, packaging safeguards, and dependency updates. Additional fixes improve static asset handling and development workflows.


Changes
-------

Backports
^^^^^^^^^

- Enforced same-origin and HTTP-only redirection when breaking from an AJAX
  content container and during normal single page application navigation.
- Added ``__pycache__`` folders to the project's ``.gitignore`` file.
- Made the Smart Link condition fields ``inclusion`` and ``operator`` fully
  translatable.
- Made document file search fields translatable.
- Made the tag document list link title translatable.
- Made the task worker labels translatable.
- Added a missing ``locales`` app migration.
- Ensured that only Mayan EDMS packages are uploaded to PyPI.


Dependencies
^^^^^^^^^^^^

- Updated Django from version 4.2.26 to 4.2.27 to address
  CVE-2025-13372 and CVE-2025-64460.
- Updated ``django-auth-ldap`` from version 4.8.0 to 5.2.0.


Packaging
^^^^^^^^^

- Added ``appearance/node_modules/bootswatch/docs/*`` to the appearance app's
  ``static_media_ignore_patterns``.


Development
^^^^^^^^^^^

- Added a Makefile target to clean up all Python libraries.
