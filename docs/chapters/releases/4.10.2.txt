Version 4.10.2
==============

Released: December 13, 2025

Summary
-------

This release delivers important security hardening for navigation redirects, updates the Docker base image, improves internationalization coverage across multiple UI components, and includes minor quality-of-life and maintenance fixes.

Changes
-------

Security
^^^^^^^^

- Enforced same-origin and HTTP-only redirection when breaking from an AJAX content container and during normal single-page application navigation.


Platform
^^^^^^^^

- Updated the base Docker image from `debian:12.11-slim` to `debian:12.12-slim`.


Code quality
^^^^^^^^^^^^

- Added `__pycache__` folders to `.gitignore`.


Internationalization
^^^^^^^^^^^^^^^^^^^^

- Made the Smart Link condition fields `inclusion` and `operator` fully translatable.
- Made document file search fields translatable.
- Made the tag document list link title translatable.
- Made the task worker labels translatable.
- Added missing locales app migration.
